Add a new commit on an already committed and pubed one 

------------------------------------------- 

git commit -a 

git pub 

Add a new patch on top of the last one 

git commit --amend 

(put change id under the ISSUE no) 

git add /path/to/the/modified/file/ 

git pub 

  

When modifications must be added to another branch rather than the current one. 

---------------------------------------------- 

If you want to keep the changes, 

------------------------------------------- 

git stash save 

git checkout branch 

git stash pop 

  

  

  

Discard all the changes on the branch 

------------------------------------------- 

git reset HEAD^ 

  

  

git checkout HEAD^ 

  

Undo uncommitted changes 

git reset --hard 

------------------------------------------- 

  

  

If all you want to do is undo the act of committing, leaving everything else intact, use: 

------------------------------------------- 

git reset --soft HEAD^ 

  

If you want to undo the act of committing and everything you'd staged, but leave the work tree (your files intact): 

------------------------------------------- 

git reset HEAD^ 

  

And if you actually want to completely undo it, throwing away all uncommitted changes, resetting everything to the previous commit (as the original question asked): 

------------------------------------------- 

git reset --hard HEAD^
