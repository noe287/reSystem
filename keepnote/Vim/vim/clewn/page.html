When cross-compiling, add the --host and --build argument on the command line of configure. Then run make. For example, with an arm target: ./configure CC=/path_to_gcc/arm-gcc --build=i686-pc-linux-gnu --host=arm-linux The readline and curses libraries must be available on the target. The following commands are run: host: gvim -c "run clewn.vim" -nb:target_ip_address target: clewn -x pathnames_map -nb:target_ip_address
