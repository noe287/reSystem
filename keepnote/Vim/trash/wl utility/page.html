WL:commands 

  

Run FW without ProxySTa 

--------------------------------- 

1. Stop ASP  

killall -SIGSTOP mngr 

killall -SIGSTOP awf-engine 

  

2. Set Modes 

wl -i wl0 down 

wl -i wl0 ap 0 

wl -i wl0 apsta 0 

wl -i wl0 psta 0 

wl -i wl0 up 

ifconfig wl0 up 

  

3. Remove from bridge 

brctl delif br0 wl0 

  

4. Connect 

wl -i wl0 up 

wl -i wl0 wsec 4 

wl -i wl0 sup_wpa 1 

wl -i wl0 set_pmk "25055184" 

wl -i wl0 roam_off 1 

wl -i wl0 join "TP-LINK_2.4GHz_D69E8D" imode bss amode wpa2psk 

  

5. DHCP Req  

udhcpc -i wl0
