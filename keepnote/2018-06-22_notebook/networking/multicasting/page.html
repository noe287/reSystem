IP TV and Multicasting 

------------------------------------------------- 

IPERF 

----------------------- 

iperf -c 224.0.22.2 -u -b 20m -i1 -t30 

iperf -s -u -i1 -B 224.0.2.2  

  

  

Add multicast traffic Route to the Route Table 

----------------------------------------------- 

sudo route add -net 224.0.0.0 netmask 240.0.0.0 dev enp5s0 

sudo route add -net 224.0.0.0/8 dev eth0 

  

  

Install pimd for igmp querrier or snooping 

-------------------------------------------- 

Edit the /etc/pimd/conf file for disabling ports to not to send querry on them. 

then run /etc/init.d/pimd start
